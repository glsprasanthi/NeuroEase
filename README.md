# NeuroEase

NeuroEase is an inclusive learning interface designed for neurodivergent users, supporting Dyslexia, ADHD, Autism, and Anxiety needs through adaptive UI and interaction modes.

## Tech Stack
- React + TypeScript
- Vite
- CSS
- Context API

## Features
- Mode-based personalization
- Dyslexia-friendly reading interface
- ADHD focus-based content flow
- Autism-friendly structured layout
- Anxiety support panel

## Setup
npm install  
npm run dev
